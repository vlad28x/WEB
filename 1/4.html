Ссылка reference — механизм языка программирования (C++), позволяющий привязать имя к значению. В частности, ссылка позволяет дать дополнительное имя переменной и передавать в функции сами переменные, а не значения переменных. 

int n = 0;
int &r = n; /* теперь r -- ссылка на n
n = 10;cout << r << '\n'; // выведет 10
r = 20;
cout << n << '\n'; // выведет 20
cout << (&n == &r) << '\n'; // выведет 1, т.е. истина

Переменная, значением которой является адрес, называется указателем pointer.
Разыменованием указателя pointer dereferencing называют обращение к значению по адресу, хранимому указателем. 

int value = 5;
int *ptr = &value; // инициализируем ptr адресом значения переменной
